# Audio-Noise-Reduction

In this project we are trying to filter out the noise from the input signal while simultaneously avoiding the degrading of the signal quality. For this purposes we are using a U-Net type convolutional autoencoder.
The model is trained on a publicly available audio dataset at Datashare. The dataset contains 11572 records in the shortÂ .wav format. It contains the clean versions of the audio records as well as the corresponding versions of the noisy audio records.
